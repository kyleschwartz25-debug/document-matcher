# QUICK START - Document Matcher

## First Time? Do This Once:

1. **Install Python 3.11+**
   - Go to https://www.python.org/downloads/
   - Download Python 3.11.x installer
   - Run installer, CHECK "Add Python to PATH"

2. **Run Setup Script**
   - Double-click `SETUP.bat` in this folder
   - Wait for it to complete

3. **Done!** You're ready to use the tool

---

## Using the Tool (30 seconds per comparison):

1. Click **SELECT SO PDF** → Pick your Sales Order PDF
2. Click **SELECT PO PDF** → Pick your Purchase Order PDF
3. Click **COMPARE** → Get instant results
4. Click **COPY TO CLIPBOARD** → Paste into CRM

**Result:** Shows ✓ COMPLETE MATCH or ✗ MISMATCH with details

---

## Files:

| File | What to Do |
|------|-----------|
| `SETUP.bat` | Run this ONCE to install Python & dependencies |
| `document_matcher.py` | Double-click THIS to launch the tool |
| `PYTHON_SETUP.md` | Detailed setup instructions (if you need help) |
| `Example_Pairs/` | Test documents (11 SO/PO pairs) |

---

## Troubleshooting:

| Problem | Solution |
|---------|----------|
| "Python not found" | Install Python from python.org, make sure to check "Add to PATH" |
| App won't start | Run `SETUP.bat` again to install dependencies |
| PDF won't load | Make sure it's a text-based PDF (not scanned image) |
