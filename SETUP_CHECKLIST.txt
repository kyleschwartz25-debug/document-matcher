## ‚ö†Ô∏è Troubleshooting

### Problem: "Python not found"
**Solution:**
- Install Python from https://www.python.org/downloads/
- Make sure to check "Add Python to PATH"
- Restart your computer
- Try again

### Problem: "PyPDF2 not found" or "ModuleNotFoundError"
**Solution:**
- Run `SETUP.bat` again
- Or manually: `python -m pip install PyPDF2`

### Problem: Application won't open
**Solution:**
- Open Command Prompt in the Document Checker folder
- Run: `python document_matcher.py`
- Look for error messages
- Check PYTHON_SETUP.md for detailed troubleshooting

### Problem: "PDF extraction failed"
**Solution:**
- Make sure the PDF is text-based (not scanned image)
- Try opening the PDF in Adobe Reader and re-saving it
- Check if file path has special characters
